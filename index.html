Portfolio

Project Terbaru

Berikut adalah koleksi proyek terbaru saya yang menampilkan keahlian dan kreativitas dalam pengembangan web. Setiap proyek mencerminkan dedikasi saya terhadap kualitas dan inovasi dalam desain dan fungsionalitas.

Website Berbasis GIS dan IoT untuk Memantau Kesuburan Tanah Kebun Raya Cibodas

Sistem ini menggunakan teknologi GIS untuk visualisasi data kesuburan tanah, memungkinkan pengguna untuk menjelajahi informasi secara interaktif. Dengan database spatial yang menyimpan data terkait kesuburan tanah, proyek ini memberikan analisis mendalam tentang kandungan NPK, pH, dan Kelembaban Tanah.

Web-GIS tentang Analisis Spasial Karakteristik Habitat Flora di Kalimantan Timur

Proyek ini fokus pada analisis spasial untuk memahami karakteristik habitat flora di wilayah Kalimantan Timur. Melibatkan pemanfaatan teknologi GIS dan data spasial, kami menyelidiki faktor-faktor lingkungan yang memengaruhi distribusi dan keberlanjutan berbagai jenis flora. Penelitian ini tidak hanya memberikan pemahaman mendalam tentang ekosistem lokal, tetapi juga mengeksplorasi potensi aplikasi teknologi GIS dalam pelestarian dan pengelolaan sumber daya alam.

Sistem Internet of Things (IoT) dan pemetaan GIS untuk meningkatkan efisiensi dalam pemberian dosis pupuk

Sistem ini bertujuan untuk memonitor secara real-time kebutuhan nutrisi tanaman, mengintegrasikan data sensor IoT untuk mendeteksi tingkat kelembaban tanah, nutrisi, dan kondisi lingkungan. Melalui antarmuka pemetaan GIS, informasi ini dapat divisualisasikan dengan jelas dan diakses dengan mudah oleh para petani.
